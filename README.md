# お父さんポイント

家族で「お父さんポイント」を加点・減点・ごほうび交換できる、単一HTMLのスマホ向けアプリです。

## 使い方

`index.html` をブラウザで開くだけです。

## 機能

- 家事、送迎、勉強を見る、ごはん作りなどをワンタップ加点
- 寝落ち、スマホ見すぎ、約束忘れなどの減点
- 自由な理由と点数で登録
- ごほうび交換
- 履歴表示
- 名前変更
- localStorage保存
- JSONバックアップ保存

## GitHub Pages

リポジトリの Settings → Pages で Branch を `main`、Folder を `/root` にすると公開できます。
