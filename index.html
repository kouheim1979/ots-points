Supabase設定

本体保存

Supabaseを設定すると、リン・ミク・OKSのデータを外部データベースで共有できます。未設定なら端末内保存です。

Supabase SQL

create table if not exists dad_points_logs ( id uuid primary key default gen_random_uuid(), member text not null, reason text not null, points integer not null, who text not null default '家族', created_at timestamptz not null default now() ); create table if not exists dad_points_rewards ( id uuid primary key default gen_random_uuid(), name text not null, cost integer not null, created_at timestamptz not null default now() ); alter table dad_points_logs enable row level security; alter table dad_points_rewards enable row level security; create policy "read logs" on dad_points_logs for select using (true); create policy "insert logs" on dad_points_logs for insert with check (true); create policy "read rewards" on dad_points_rewards for select using (true); create policy "insert rewards" on dad_points_rewards for insert with check (true);

service_role key は入れないでください。anon public keyだけ使います。
